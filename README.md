# Wallet manager API
## Script to start the server

###`npm run dev`

 After running this script, the server start on the port `8081`
## Endpoints
* ###`post /wallet`
  ###Requirement
  * ### uid : 32 digit alphanumeric string
  * ### secret : alphanumeric string
* ### `get /wallet`
  ###Requirement
    * ### uid : 32 digit alphanumeric string
    * ### secret : alphanumeric string
